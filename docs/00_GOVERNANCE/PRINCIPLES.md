# Principles

Version: 1.0
Status: Approved

## Core Principles

1. Decisions over data.
2. Trust first.
3. Today first.
4. Simplicity wins.
5. One question per screen.
6. Explain every recommendation.
7. Mobile first.
8. Depth before breadth.

## Practical Meaning

MoneyPulse should provide practical answers before detailed reports.

The MVP should focus on daily clarity, purchase simulation and goal impact.
