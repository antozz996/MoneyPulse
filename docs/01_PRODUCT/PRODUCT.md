# Product

Version: 1.0
Status: Draft
Owner: Product

## Vision

MoneyPulse becomes the most trusted daily decision layer for personal money.

## Product Question

How much can I safely spend today, and what happens if I spend more?

## Primary User

An everyday consumer who has income, recurring obligations, and financial goals, but does not want to manage life through spreadsheets.

## Jobs To Be Done

- Help me understand how safe today feels financially.
- Help me test a purchase before I commit.
- Help me see what future tradeoff a decision creates.
- Help me improve without overwhelming me.

## Core Experiences

### Today

The landing experience shows a daily safe-to-spend recommendation, upcoming pressure, and the main reasons behind the recommendation.

### Before You Buy

The user enters a possible expense and immediately sees whether it is safe, risky, or harmful, plus what changes tomorrow and this month.

### Money

The user can inspect the accounts, cash flow, and commitments powering the recommendation.

### Goals

The user sees how discretionary spending affects short- and medium-term goals.

### Insights

The product surfaces a few useful behavioral patterns instead of generic charts.

## MVP Scope

The MVP must support:

- manual financial inputs;
- daily affordability calculation;
- purchase simulation;
- explainable outputs;
- mobile-first experience.

## Out Of Scope For MVP

- bank aggregation at scale;
- investment tracking;
- budgeting categories with heavy configuration;
- household multi-user collaboration;
- tax planning.

## Success Metrics

- Time to first answer under 30 seconds.
- User can understand safe-to-spend number without help.
- Purchase simulation returns a recommendation instantly.
- Weekly active users return because the answer changes with real context.

## Product Release Rule

No feature ships unless it improves one of the core experiences and can be expressed in plain language.
